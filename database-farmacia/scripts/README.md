# Scripts de database-farmacia

Esta carpeta debe contener scripts auxiliares del componente de base de datos de farmacia.

## Usos esperados

- creación o recreación controlada del esquema;
- seeds de demostración;
- backups o restauración local;
- consultas de revisión de catálogo, publicación o disponibilidad.

## Reglas

- nombres explícitos;
- sin credenciales reales;
- comentarios mínimos cuando el script tenga riesgo o varias etapas;
- evitar basura temporal sin propósito claro.

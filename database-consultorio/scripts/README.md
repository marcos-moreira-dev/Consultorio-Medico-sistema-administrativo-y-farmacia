# Scripts de database-consultorio

Esta carpeta debe contener scripts auxiliares del componente de base de datos del consultorio.

## Ejemplos razonables

- creación o recreación controlada del esquema;
- carga de seeds de estudio;
- backups locales;
- consultas auxiliares de revisión;
- utilidades para ambientes de desarrollo o demo.

## Reglas

- no subir credenciales;
- no meter dumps pesados sin justificación;
- documentar brevemente scripts delicados;
- mantener nombres claros y específicos.

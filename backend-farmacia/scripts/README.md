# Scripts de backend-farmacia

Esta carpeta debe contener utilidades auxiliares del componente `backend-farmacia`.

## Usos esperados

- arranque local de desarrollo;
- tareas repetitivas del entorno Node/Nest;
- ayudas para revisión, carga o saneamiento controlado;
- comandos de soporte que no merecen estar enterrados en un chat.

## Comandos del proyecto que conviene conocer primero

- `npm run build`
- `npm run typecheck`
- `npm run test:unit`
- `npm run test:e2e -- --runInBand`
- `npm run verify`

## Reglas

- nombres explícitos;
- comentarios mínimos si el script no es obvio;
- nada de secretos;
- nada de archivos desechables sin propósito.

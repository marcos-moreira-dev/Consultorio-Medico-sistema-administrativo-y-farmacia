# Scripts de storefront-farmacia-angular

Esta carpeta contiene scripts auxiliares del storefront Angular pensados sobre todo para desarrollo local en Windows/PowerShell.

## Scripts previstos en esta tanda

- `dev.ps1`: arranca el frontend en modo desarrollo.
- `build.ps1`: ejecuta el build del storefront.
- `test.ps1`: corre la suite base de tests.
- `verify.ps1`: corre typecheck y build.

## Regla práctica

Los scripts de esta carpeta son envoltorios de conveniencia. La fuente oficial de verdad sigue siendo `package.json`.
